ZeroSkid: Wet-Road Detection with AlexNet

ZeroSkid is an assistive driving system that uses computer vision and deep learning to detect wet pavement in real time. Wet roads cause thousands of preventable crashes every year. This project warns drivers before traction is lost.

Problem Solution Technical Training Visualizations Takeaways

Problem & Motivation

Wet pavement is a serious but often overlooked hazard. It contributes to hundreds of thousands of crashes each year and leads to major financial and emotional costs.

  • ABS and traction control only react after a skid starts.
  • Drivers can’t always see thin water layers or puddles, especially at night.
  • We need proactive, real-time detection — not reactive correction.

ZeroSkid Solution

ZeroSkid uses a dash-mounted camera and an embedded deep learning model to classify road conditions in real time.

  • Camera-based sensing
  • Wet vs. dry road classification
  • Instant alerts for drivers

Technical Approach: AlexNet

AlexNet is a deep convolutional neural network that learns textures, reflections, and patterns that appear on wet pavement.

Architecture Summary

  • 5 convolutional layers
  • 3 fully connected layers
  • ReLU activations & max pooling
  • Softmax output for wet/dry classification

Training & Data

Training Setup

Input: Wet & dry road images
Loss: Cross-entropy
Optimizer: SGD or Adam
LR: 0.001
Batch size: 32
Epochs: 10–30

Preprocessing

  • Resize to 224×224×3 (AlexNet standard)
  • Normalize pixel values
  • Brightness/rotation/flip augmentation
