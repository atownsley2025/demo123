ZeroSkid

Using AI and computer vision to detect wet roads in real-time and prevent accidents before they happen.

Problem & Solution

Wet pavement causes thousands of preventable crashes every year. Most vehicle systems react *after* a skid begins. ZeroSkid aims to fix this with proactive detection using deep learning.

  • Detects slippery road conditions instantly
  • Warns drivers before traction loss
  • Affordable dash-mounted AI system

Tools & Tech

This project uses AlexNet, computer vision, and embedded hardware.

  • Python
  • PyTorch / TensorFlow
  • AlexNet CNN architecture
  • Image preprocessing + augmentation

Data, Training & Results

Model trained on wet vs dry road images with augmentation.

  • Learning rate: 0.001
  • Batch size: 32
  • Loss: Cross-entropy
