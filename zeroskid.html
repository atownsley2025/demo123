Tools & Technical Approach

Here you’ll describe AlexNet, how the architecture works, and what tools you’re using (Python, PyTorch / TensorFlow, Google Colab, etc.).

AlexNet Overview

  • Deep convolutional neural network for image classification.
  • 5 convolutional layers and 3 fully connected layers.
  • Great at learning textures, reflections, and patterns on wet pavement.

Training Setup

Input images: Wet + dry road images
Loss: Cross-entropy
Optimizer: SGD or Adam
Learning rate: 0.001
Batch size: 32
Epochs: 10–30
