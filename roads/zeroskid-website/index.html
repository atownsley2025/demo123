# ZeroSkid ‚Äì Wet vs. Dry Road Classifier ## Problem & Motivation Hydroplaning accidents happen fast and often because drivers can't always tell how wet a road really is. My goal with ZeroSkid is to build a simple AI model that can classify road conditions as *wet* or *dry* so drivers and vehicle systems can react early. ## Dataset Description I created my own dataset of wet and dry road images. The images came from online sources and were organized into two folders: - `wet/` - `dry/` All images were resized to 224√ó224 and normalized for AlexNet. Because my dataset is small, this project focuses on testing how well AlexNet can learn from limited data and how different settings affect performance. ## Technical Approach (AlexNet) I used PyTorch in Google Colab and implemented a modified AlexNet model trained on my custom dataset. The process included: - Loading data from Google Drive - Random horizontal flips for augmentation - Normalization for AlexNet - Training for 20+ epochs - Running multiple experiments: - different batch sizes - different learning rates - with vs. without augmentation - AlexNet vs. ResNet18 All experiments were logged using Weights & Biases. --- # üß™ Experiments & Results ## Experiment 1 ‚Äì Baseline This experiment used: - Batch size 16 - Learning rate 0.001 - No augmentation ### Accuracy ![Experiment 1 Accuracy](images/exp1_accuracy.png) ### Loss ![Experiment 1 Loss](images/exp1_loss.png) --- ## Experiment 3 ‚Äì Best Performance This experiment performed the best.
